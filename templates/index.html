Customizable

This plugin is highly customizable . Change colors, margins and fonts easily via css. The plugin comes packed with six modes that can be changed via HTML.

Responsive

This plugin is ultra responsive. It resizes from mobile to HD easily. Works great with touch devices and uses css media query to fit on any device.

Modular

Six Modes to choose from. All with the same structure, just change the mode to any of mode-default, mode-oncenter, mode-slider, mode-yearslist, mode-blackwhite or mode-masonry to satisfy any creative need.


SEO Friendly

This plugin uses standard crawlable html to display the content. The use of headings and meta tags make it more search engine friendly.

HTML5 and CSS3

The latest use in html technology makes this plugin up to today standards. CSS3 animations and html5 tags are used throughout the plugin.

CSS Only

No javascript dependency, no jquery library, just works everywhere you put it without no processing power taken.
